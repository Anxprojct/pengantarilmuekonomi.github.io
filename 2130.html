Perusahaan konveksi mengeluarkan biaya tetap Rp 1.000,-, biaya variabel Rp 750,- biaya pemasaran 60% dari total biaya penjualan produk mencapai 2000 satuan dengan harga jual Rp 2 500,-. Maka perusahaan rugi sebesar....
       
Hint
Untuk menjawab pertanyaan ini Anda harusbelajar lbih dahulu modul 5.
A. Rp. 400.000,00
B. Rp. 500.000,00
C. Rp. 600.000,00
D. Rp. 700.000,00

A. salah, karena perhitungan yang benar: penjualan 2000 x 2.500= 5.000.000 – 5.600.000 = 600.000

(2000 x 1000) + (2000 x 750) + {60% x {(2000 x 1000) + (2000 x 750)}}= 5.600.000

B. salah, karena perhitungan yang benar: penjualan 2000 x 2.500= 5.000.000 – 5.600.000 = 600.000

(2000 x 1000) + (2000 x 750) + {60% x {(2000 x 1000) + (2000 x 750)}}= 5.600.000

C. benar, perhitungan penjualan 2000 x 2.500= 5.000.000 – 5.600.000 = 600.000

(2000 x 1000) + (2000 x 750) + {60% x {(2000 x 1000) + (2000 x 750)}}= 5.600.000

D. salah, karena perhitungan yang benar: penjualan 2000 x 2.500= 5.000.000 – 5.600.000 = 600.000

(2000 x 1000) + (2000 x 750) + {60% x {(2000 x 1000) + (2000 x 750)}}= 5.600.000
